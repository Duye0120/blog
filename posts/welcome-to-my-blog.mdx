## 关于这个博客

<Callout type="info">
这个博客使用了最新的 Next.js 14 和 App Router，支持服务端渲染和静态生成。
</Callout>

### 技术栈

- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS  
- **内容**: MDX (Markdown + JSX)
- **部署**: Vercel

### 为什么选择 MDX？

MDX 的强大之处在于可以在 Markdown 中嵌入 React 组件：

```typescript
// 这是代码高亮示例
const greeting = "Hello MDX!"
console.log(greeting)
```

<Callout type="success">
MDX 让我可以创建交互式内容，而不仅仅是静态文本！
</Callout>
